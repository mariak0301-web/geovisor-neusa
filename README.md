# Geovisor NEUSA

Geovisor HTML interactivo para consulta del proyecto NEUSA.

## Publicacion

El archivo principal es `index.html`, generado a partir de:

`D:\OneDrive\0_OPTIMIZACION\NEUSA\INF_GEN\HTML\geovisor_neusa.html`

Al publicarse con GitHub Pages, el geovisor abre directamente desde la raiz del sitio.
